Supported Parameters
#

  • model (required): Path to Blockbench geometry JSON
  • texture (optional): Path to texture PNG/JPG
  • width (optional): Canvas width (default: 100%)
  • height (optional): Canvas height (default: 600px)
  • autoRotate (optional): Auto-rotate toggle (default: true)
  • scale (optional): Model scale multiplier (default: 1)

Model Format
#

Supports Minecraft Blockbench geometry format (1.12.0+):

  • Cube geometry with inflation
  • Bone structure and parenting
  • Pivot points and rotations
  • UV mapping
  • Texture mapping
