# Minecraft 26.2 Snapshot 2

>Here is a surprise second snapshot for 26.2! It fixes some serious issues and crashes of the previous one released earlier this week.
>
>Happy mining!

## Changes

### Performance Improvements

-   Upgrading certain worlds from before 26.1 can now be faster

## Technical Changes

-   The `fps` F3 screen entry will now report which type of present mode is used (e.g. `mailbox` or `fifo relaxed`), as opposed to just `vsync`

## Fixed bugs in 26.2 Snapshot 2

-   [MC-306840](https://mojira.dev/MC-306840) - Dolphins still drown themselves when no players are nearby
-   [MC-307286](https://mojira.dev/MC-307286) - The game process does not terminate after exiting the game
-   [MC-307296](https://mojira.dev/MC-307296) - Launching the game with an incompatible resource pack that modifies core shaders causes the game to softlock
-   [MC-307311](https://mojira.dev/MC-307311) - Core shaders only load after restart when using Vulkan
-   [MC-307320](https://mojira.dev/MC-307320) - Vulkan prevents the game from opening and crashes some GPUs
-   [MC-307349](https://mojira.dev/MC-307349) - The game crashes before opening
-   [MC-307383](https://mojira.dev/MC-307383) - The "Exclusive Fullscreen" setting does not work with the Vulkan rendering backend
